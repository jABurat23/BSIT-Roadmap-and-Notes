# 🖥️ VS Code Setup for BSIT

> Extensions and settings that make coding life easier.

> ⚠️ **Disclaimer:** These are not the only extensions out there —
> VS Code has thousands you can explore on your own inside the Extensions
> tab (`Ctrl+Shift+X`). This list is just what *I* personally find useful
> and essential for my BSIT journey. Your setup might look different, and
> that's totally fine!
---

## 🔌 Essential Extensions

| Extension | Why you need it |
|-----------|-----------------|
| `Prettier` | Auto-formats your code so it looks clean |
| `GitLens` | See who changed what and when, right in VS Code |
| `GitHub Copilot` | AI assistant that suggests code as you type |
| `Live Server` | Instantly preview HTML files in your browser |
| `ESLint` | Catches JavaScript errors before you run the code |

## 🎓 Subject-Specific Extensions

| Extension | Subject it helps with |
|-----------|----------------------|
| `Python` | Programming, Data Structures, AI subjects |
| `C/C++` | Computer Organization, some Programming subjects |
| `MySQL Syntax` | Database Management subjects |
| `Thunder Client` | Web Development, APIs |
| `Remote - SSH` | Networking subjects |

## ⌨️ Keyboard Shortcuts I Use Daily

| Shortcut | What it does |
|----------|--------------|
| `Ctrl+Shift+X` | Open Extensions tab |
| `Ctrl+Shift+P` | Open Command Palette (search any VS Code action) |
| `Ctrl+`` ` `` | Open integrated terminal |
| `Ctrl+/` | Comment/uncomment a line |
| `Alt+Up/Down` | Move a line up or down |
| `Ctrl+D` | Select next occurrence of a word |
| `Ctrl+Shift+K` | Delete entire line |
